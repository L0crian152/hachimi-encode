# 🚀哈基米编码

## 安装步骤

```bash
# 克隆仓库
git clone https://github.com/L0crian152/hachimi-encode.git
cd hachimi-encode

# 安装依赖
pip install -r requirements.txt
```

---

## 🔧 基础命令

### 文本编码

```bash
python3 hachimi.py encode "text"
```
